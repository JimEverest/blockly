DobotLink v5.1

注意事项：

  • 有关Keyword:带有*号的为可选参数，如不传参数，将按照默认配置执行。
  • 适配Windows / MacOS系统
  • 每台上位机只能运行一个且只有一个DobotLink
  • 支持多机，支持多终端
  • 符合JsonRPC2.0传输协议
  • 采用Websocket通讯，端口9090
  • 整形，浮点型等数值如不给出将默认为0，布尔值默认为false
  • 默认日志存放路径："C:\Users\{USER}\AppData\Roaming\DobotLink\"
  • 默认传输临时文件夹："C:\Users\{USER}\dobotlink-code"
  • 驱动安装与下载：CP210 https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers


里程碑：

  • v1.0 支持插件加载，Magician控制，Microbit控制
  • v2.0 支持多机多终端控制，支持Arduino编译与下载
  • v3.0 支持最新异步动态库，合并设备至统一动态库，新增设备MagicanLite、MagicBox、M1控制
  • v4.0 新增检查更新，在线更新功能，插件热更新
  • v5.0 支持CR5协作机器人控制，SA、SR系列工业机器人控制
